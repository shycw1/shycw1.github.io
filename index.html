Doppler Radar Speed Detector

Developed a Doppler-radar based speed detector that contains a custom defined signal conditioning circuit, a programmed micro-controller that calculated FFT at 50 Hz to derive vehicle speed. The speed data was tranmitted via RS-485 to an customized interface for further processing using CPLD chip. RTL design via VHDL was implemented to map the serial data to parallel for driving the 7-segment display.
